# 简介
`MoreDrawingColor` 是一个杀戮尖塔 2 的模组，顾名思义，它能让你在大地图上绘画的时候可以使用任意颜色。

# 安装
1. 在游戏根目录创建 mods 文件夹
2. 下载本 release 的 MoreDrawingColor.zip
3. 解压后将全部文件复制到 mods 文件夹
4. 启动游戏

# 存档提醒
1. 首次启用模组时，游戏可能会弹出一次 mod 警告
2. 启用模组后，游戏可能会使用单独的 modded 存档路径
3. 如果 Steam 提示云存档冲突，请先确认本地文件状态再做选择

# Introduction
`MoreDrawingColor` is a mod for Slay the Spire 2. As the name suggests, it allows you to draw with custom colors on the map screen.

# Installation
1. Create a mods folder in your game root directory
2. Download MoreDrawingColor.zip from this release
3. Extract and copy all files to the mods folder
4. Launch the game

# Save File Notice
1. When enabling a mod for the first time, the game may show a mod warning
2. After enabling mods, the game may use a separate "modded" save path
3. If Steam prompts about cloud save conflicts, please verify your local file status before choosing
